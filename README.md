# Sreehari | Systems Security Researcher

Personal portfolio and blog built with Astro and tailored for a "Systems/Kernel Engineer" aesthetic.

## 🚀 Features

- **eBPF Terminal Simulator**: Live kernel trace visualization on the home page.
- **Bento Box Layout**: Efficiently organized projects and focus areas.
- **Pixperk-inspired UI**: Centered typography, floating pill navigation, and floating social sidebar.
- **Deep Tech Aesthetic**: Minimalist dark mode with cyan accents and grain/metal textures.
- **High Performance**: Built with Astro 5 and Tailwind CSS 4 for near-zero runtime JS.

## 💻 Tech Stack

- **Framework**: [Astro](https://astro.build/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Typography**: Instrument Serif, Space Grotesk, IBM Plex Mono
- **Deployment**: GitHub Pages (Automated via GitHub Actions)

## 🛠️ Commands

| Command | Action |
| :--- | :--- |
| `pnpm install` | Install dependencies |
| `pnpm run dev` | Start development server |
| `pnpm run build` | Build for production |
| `pnpm run preview` | Preview production build |
